# TradeSimX - Проект симуляции торговли на финансовых рынках

TradeSimX - это проект с открытым исходным кодом, разработанный для симуляции торговли на финансовых рынках. Проект включает в себя классы и методы для управления портфелем инвестора и симулирования операций покупки и продажи активов.

## Использование

Для использования TradeSimX в своем проекте, вы можете выполнить следующие шаги:

1. Склонируйте репозиторий:

git clone https://github.com/BekovaDeniza/TradeSimX

2. Тестирование

Проект включает в себя модульные тесты для проверки функциональности. Вы можете запустить тесты с помощью модуля unittest:

python test_portfolio.py
